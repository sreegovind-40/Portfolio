Goal: Build a solid, performant base that can grow with you

1. Performance Scalability HIGH

Target: Load time under 2 seconds, Lighthouse score 90+

  • Minify CSS and JavaScript files -30% file size

  • Set up version control (Git + GitHub)

3. Content Structure MEDIUM

  • Design a flexible skills grid that can accommodate new skills
  • Build a blog/articles section for future content
  • Add a testimonials section (prepare for future recommendations)

4. Hosting & Deployment HIGH

GitHub Pages Free, fast, supports custom domains
Netlify Auto-deploy, form handling, edge functions
Vercel Optimized for speed, analytics included
  • Deploy to chosen platform with automatic deployments
  • Set up custom domain (sreegovind.dev recommended)
  • Enable HTTPS
  • Configure DNS properly

1. Advanced Performance HIGH
